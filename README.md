Projeto voluntário desenvolvido como proposta da comunidade DevSquad.

## Tecnologias

- Java 21
- Spring Framework

    - Spring Security
    - Spring OpenAPI (Swagger)
- Lombok
- JUnit & Mockito
- PostgreSQL / H2 Database / Redis

- Docker & Docker compose
